Get Started

Waizatsu is a lightweight, easy-to-use Javascript library for garbling text.

Installing

ES6 Module support will be coming once it's supported by all major browsers (soon!).
For now, Waizatsu needs to be included via a CDN: 

<script src="https://linktowaizatsu.com/waizatsu.js"></script>

Hello World

See the Pen bvKMNd by Liam (@liamelgie) on CodePen.

First, create an instance of Waizatsu: 

const input = "Hello World";
const garbler = new TextGarbler(input);

This instance immediately garbles its input and assigns it to the property 'value': 

console.log(garbler.value);
=> "aDkqO lCbqW"

We can garble the input again by calling the `garble` method: 

garbler.garble();
console.log(garbler.value);
=> "LafsK MpQJw"

Repeater

Waizatsu's repeater feature allows you to repeatedly garble a string of text and transition it back to its original value on demand.
To start the repeater, call the `repeater.start` method: 

// Start the repeater
garbler.repeater.start();
// Check the garbled value
console.log(garbler.value);
=> "msAuK pWAsC"
// Check the value again to see that it has changed
console.log(garbler.value);
=> "KwiSH AKvys"

To stop the repeater, call the `repeater.stop` method. Once called, the garbled value will transition to its original value: 

// Check the garbled value
console.log(garbler.value);
=> "uJSYe AksUa"
// Stop the repeater
garbler.repeater.stop();
// Check the value again to see that it has transitioned
console.log(garbler.value);
=> "Hello World"
