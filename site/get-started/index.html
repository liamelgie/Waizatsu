Get Started

Waizatsu is a lightweight, easy-to-use Javascript library for garbling text.

Installing

ES6 Module support will be coming once it's supported by all major browsers (soon!).
Until then, Waizatsu will need to be included via a CDN: 

<script src="https:linktowaizatsu.com/waizatsu.js"></script>

Hello World

First, create a TextGarbler object: 

const input = "Hello World";
const garbler = new TextGarbler(input);

As soon as the object is created, it will garble its input and store it in the 'value' property: 

console.log(garbler.value);
=> "aDkqO lCbqW"

We can garble the input again by calling the object's `garble` method: 

garbler.garble();
console.log(garbler.value);
=> "LafsK MpQJw"

This is the most basic possible operation and it's much use.
To find out what you can do with this library, take a look at the documentation and check out some of the examples.
